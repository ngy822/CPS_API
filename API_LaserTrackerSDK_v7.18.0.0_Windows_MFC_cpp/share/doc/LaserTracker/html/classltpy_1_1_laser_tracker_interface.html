◆ flip_frontsight_backsight()

ErrorType ltpy.LaserTrackerInterface.flip_frontsight_backsight (   self)

Function to flip the Laser tracker gimbal between frontsight and backsight.

The gimbal will be flipped to other side i.e., $$ az_{target} = 180 \pm az_{current} $$ $$ el_{target} = 180 - el_{current} $$

Note
This function call will only flip the gimbal. It will not perform any measurements If the function returns ErrorType::Success, TaskMode::FlipFSBS is assigned and a callback is sent when the task completes.
Returns
ErrorType::Success If the task is successfully started.
ErrorType::DeviceNotConnected If device is not connected.
ErrorType::OtherTaskOnGoing If the device is performing another task.
