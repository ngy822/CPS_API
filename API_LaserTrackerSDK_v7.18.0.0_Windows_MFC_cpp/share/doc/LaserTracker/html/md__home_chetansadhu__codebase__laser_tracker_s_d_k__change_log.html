Release Notes

🏷️ Version 7.18.0.0

🗓️ Release Date 23 March 2026

✨ New

  • Added Get6DSinglePointMeasurement(...) overloaded function with the option to get 4x4 homogeneous matrix.
  • Added PointToTrackerAndSts(...) to move both STS and Tracker togther.
  • Added material_temperature1 through material_temperature3 to WeatherStationInfo struct. This is currently supported for RadianPro and RadianPlus.

🔄 Changes

  • Introduced pure jogging of the gimbal for iLT similar to our Radian Tracker Models.

🐛 Bugfixes

  • Fixed TTL Trigger Data missing when Trigger Counter goes to 0

🏷️ Version 7.17.2.0

🗓️ Release Date 13 February 2026

✨ New

  • Added new sample codes for C++ in addition to the MFC based sample code

🔄 Changes

  • Changed the gimbal wrap-around behavior to avoid gimbal hitting to the stopper
  • Changed Frontsight Backsight Check window to include Current Tracker AZ and EL positions. Also added realtime AZ and EL display.

🏷️ Version 7.17.1.0

🗓️ Release Date 15 January 2026

🔄 Changes

  • Added ability to change error tolerance and shake tolerance settings during stylus calibration. Also, the error tolerance will automatically adjust based on the length of the stylus.
  • Added ability to Edit stylus ID even though the stylus is locked.
  • Added option to delete the calibration point on the right click during the stylus calibration.
  • Removed the prompt during the iVision window closing.
  • Added shortcut to iVision jogging in the iVision Control Window. If jog is enabled, Keyboard Arrow Keys can be used to jog the gimbal.
  • Changed error return message to include accessory prm file name when the file is not found.

🐛 Bugfixes

  • Fixed SMR temporal scanning time period adding 1 extra millisecond.
  • Fixed Laser beam pointing off the target when FrontsightBacksight operation is used after virtual level.

  • Fixed battery status always showing 100% for iLT.

🏷️ Version 7.16.0.0
